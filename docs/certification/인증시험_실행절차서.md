## 1. 시험 환경 준비

### 1.0 시험 환경 안내

**중요**: 본 인증시험은 **완전 자동화된 소프트웨어 시험**입니다.

- ✅ **필요한 것**: Windows PC, Python 환경
- ❌ **불필요한 것**: 실제 PLC, VFD, 센서 등 외부 하드웨어

**시뮬레이션 방식**:
- PLC 통신: Mock Modbus 클라이언트 (내장)
- VFD 제어: Mock VFD 시뮬레이터 (내장)
- 센서 데이터: 시나리오 기반 자동 생성 (내장)
- HMI 렌더링: Mock HMI 렌더러 (내장)

**실제 운영 환경**:
- Edge AI 장치: NVIDIA Jetson Xavier NX
- PLC: Siemens S7-1200 (Modbus TCP)
- VFD: Danfoss VLT AutomationDrive FC302

→ **시험 시에는 위 실제 장비 불필요**, 모든 것이 소프트웨어로 시뮬레이션됩니다.

### 1.1 하드웨어 요구사항

- **시험 PC**: Windows 10 이상
- **CPU**: 일반 데스크톱 CPU (특별한 요구사항 없음)
- **메모리**: 4GB 이상
- **저장공간**: 1GB 이상 여유 공간

## 2. 시험 항목 1: PLC 제어 응답속도

### 2.1 시험 개요

**측정 대상**: AI 계산 완료 → VFD 변경 완료 시간
**측정 횟수**: 50회 (50개 서로 다른 시나리오)
**합격 기준**:
- 평균 응답시간: 0.6~0.8초
- 최대 응답시간: <1.0초

### 2.2 시험 절차

#### Step 1: 작업 디렉토리 이동

```bash
cd C:\Users\my\Desktop\EDGE_AI_REAL
```

#### Step 2: 시험 스크립트 실행

```bash
python tests\test_plc_response_time.py
```

#### Step 3: 시험 진행 확인

시험이 시작되면 다음과 같이 출력됩니다:

## 6. 문제 해결 (Troubleshooting)

### 6.1 Python 실행 오류

**증상**: `python` 명령을 인식하지 못함

**해결**:
```bash
# Python 경로 확인
where python

# 또는 python3 시도
python3 tests\test_plc_response_time.py
```

### 6.2 라이브러리 없음 오류

**증상**: `ModuleNotFoundError: No module named 'numpy'`

**해결**:
```bash
pip install numpy pandas psutil
```

### 6.3 인코딩 오류

**증상**: `UnicodeEncodeError`

**해결**: 이미 스크립트에서 처리되어 있으므로 문제없음

### 6.4 폴더 경로 오류

**증상**: `FileNotFoundError`

**해결**:
```bash
# 현재 경로 확인
cd

# 올바른 경로로 이동
cd C:\Users\my\Desktop\EDGE_AI_REAL
# 또는
cd C:\Users\my\Desktop\HMI_REAL
```
